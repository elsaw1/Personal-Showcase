Tapi, kalau si introvert yang keluar, jangankan buat ngobrol, ke tempat rame dikit aja udah lemes.

Apa itu Pemrograman Web?

Pemrograman Web merupakan mata kuliah yang mempelajari mengenai pemrograman frontend hingga backend suatu website.

HTML & CSS

Mempelajari dasar-dasar struktur dan styling website menggunakan HTML dan CSS.

JavaScript

Mengimplementasikan interaktivitas dan logika pemrograman pada website.

Backend Development

Membangun server, API, dan mengelola database untuk aplikasi web.

Mata kuliah Pemrograman Web memberikan kami pemahaman mendalam tentang pengembangan website modern, mulai dari frontend hingga backend, serta prinsip-prinsip desain yang responsif dan user-friendly.
